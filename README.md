## Folder Structure
Here is the core folder structure which flutter provides.

```
flutter-app/
|- android
|- build
|- ios
|- lib
|- test
```
Here is the folder structure we have been using in this project

```
lib/
|- src/
  |- modules/
  |- shared/
  |- app_widget.dart
|- generated_plugin_registrant.dart
|- main.dart
```
